# Roadmap

## Version 1.0.0

Initial DevOps workflow simulation.

- Go API service
- Docker deployment
- Nginx reverse proxy
- Jenkins pipeline
- GitHub Actions CI
- Health check and backup automation
- Operational documentation

## Future Improvements

### Infrastructure

- Deploy to a real cloud VPS
- Add HTTPS using Let's Encrypt
- Add domain-based Nginx configuration
- Add environment separation: development, staging, production

### Monitoring

- Add Prometheus metrics
- Add Grafana dashboard
- Add alert notification using Telegram or email

### Security

- Add Docker image vulnerability scanning
- Add secret management
- Add Nginx access log analysis

### CI/CD

- Add automated rollback
- Add deployment approval stage
- Push Docker image to container registry
