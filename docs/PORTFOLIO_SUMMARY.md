# InfraWatch - Project Summary

InfraWatch is a practical DevOps operations project built with Go, Docker, Nginx, Jenkins, GitHub Actions, and shell scripting.

This project demonstrates:

- Go API service deployment
- Docker and Docker Compose orchestration
- Nginx reverse proxy configuration
- Jenkins CI/CD pipeline
- GitHub Actions CI workflow
- Health check automation
- Backup automation
- Log and status scripts
- Security hardening with rate limiting and non-root container user
- Operational SOP documentation

The project simulates real DevOps responsibilities such as deployment, monitoring, incident response, backup and recovery, and production readiness validation.
